# Web Security Engineer Skill Sheet
# Web安全工程师技能表

## 适用人群
  未来准备从事Web安全相关工作的人群；
  希望学习Web安全技能的初学者；
  每一位网络信息安全爱好者。

## 学习路线
### Web安全相关概念
1.熟悉基本概念  
熟悉SQL注入、上传、XSS、CSRF、一句话木马等基本概念；
2.熟悉渗透工具  
熟悉AWVS、SQLMap、Burp、Nessus、NMap、chopper等工具的使用方法；
3.熟悉Windows/Kali操作系统  
熟悉Windows/Kali linux操作系统的基本命令、常用工具；
4.渗透实战操作  
掌握渗透的整个阶段并能够独立渗透小型站点。
